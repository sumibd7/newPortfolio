  • CSS:CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS controls the layout, colors, fonts, and overall design of web pages, allowing developers to separate content (HTML) from design (CSS). This ensures a clean structure, reusability, and easier maintenance.
  • JAVAScript:JavaScript is a powerful, lightweight programming language commonly used to create interactive and dynamic content on web pages. It is a client-side language (executed in the user's browser) but can also be used on the server side with environments like Node.js. JavaScript plays a critical role in modern web development, providing the logic and behavior to complement HTML's structure and CSS's design
  • BOOTSTRAP:Bootstrap is a popular open-source front-end framework used to create responsive, mobile-first websites.
