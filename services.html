HTML

HTML (HyperText Markup Language) is the fundamental language used to create and design web pages. It provides the structure and layout of a webpage by organizing content like text, images, links, forms, and multimedia elements

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS controls the layout, colors, fonts, and overall design of web pages, allowing developers to separate content (HTML) from design (CSS). This ensures a clean structure, reusability, and easier maintenance.

BOOTSTRAP

Bootstrap is a popular open-source front-end framework used to create responsive, mobile-first websites. It includes HTML, CSS, and JavaScript-based components for designing web pages efficiently. Bootstrap simplifies the process of building websites by providing pre-designed components and a grid system, reducing the need to write custom CSS

JAVAScript

JavaScript is a powerful, lightweight programming language commonly used to create interactive and dynamic content on web pages. It is a client-side language (executed in the user's browser) but can also be used on the server side with environments like Node.js. JavaScript plays a critical role in modern web development, providing the logic and behavior to complement HTML's structure and CSS's design
